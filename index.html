HTML as One of My Core Skills

HTML stands for HyperText Markup Language. It is a markup language used to structure web content. It does not style content but defines the structure and meaning of elements on a webpage.

Structural Elements

Element What It Is Use
<html> Root element Wraps entire document
<head> Metadata container Stores title and meta information
<body> Main content container Displays webpage content

Text Formatting Elements

<strong> is used for important text.

<em> is used to emphasize text.

Lists

<ul> creates unordered lists.

<ol> creates ordered lists.

Links

<a> is used to create hyperlinks between pages or sections.

Images

<img> embeds images and requires src and alt attributes.
